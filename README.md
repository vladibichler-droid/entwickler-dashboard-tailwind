# Entwickler Dashboard

Ein modernes Entwickler-Dashboard mit TypeScript und Tailwind CSS.

## Funktionen

- Projekte anlegen
- Projekte löschen
- Projektstatus ändern
- Fortschrittsbalken
- Projektstatistik
- Browser-Speicherung (LocalStorage)
- Projektfilter

## Verwendete Technologien

- HTML
- TypeScript
- Tailwind CSS

## Versionen

- Version 1 bis Version 10 erfolgreich umgesetzt
