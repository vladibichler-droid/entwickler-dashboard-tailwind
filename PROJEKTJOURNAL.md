# PROJEKTJOURNAL

## Projekt
Entwickler Dashboard

## Ziel
Ein modernes Dashboard zur Verwaltung persönlicher Programmierprojekte.

## Verwendete Technologien
- HTML
- TypeScript
- Tailwind CSS

## Entwicklungsverlauf

### Version 1
Grundgerüst erstellt.

### Version 2
Projektkarten eingeführt.

### Version 3
Status farbig dargestellt.

### Version 4
Neue Projekte können angelegt werden.

### Version 5
Löschfunktion ergänzt.

### Version 6
Projektstatus kann geändert werden.

### Version 7
Fortschrittsbalken eingebaut.

### Version 8
Projektstatistik ergänzt.

### Version 9
LocalStorage integriert.

### Version 10
Projektfilter hinzugefügt.

## Ergebnis

Das Dashboard ermöglicht das Anlegen, Verwalten, Filtern und Speichern von Projekten direkt im Browser.
